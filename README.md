# AL-MO Portfolio

بورتفوليو ثنائي اللغة (عربي / إنجليزي) مبني بـ React + TypeScript + Vite + Tailwind + Framer Motion.

## التشغيل محليًا

```bash
npm install
npm run dev
```

يفتح الموقع على `http://localhost:5173`.

للبناء الإنتاجي:

```bash
npm run build
npm run preview
```

## البنية

```
src/
  i18n/translations.ts     ← كل نصوص الموقع (عربي وإنجليزي) في مكان واحد
  context/LanguageContext  ← إدارة اللغة و RTL/LTR وحفظها في localStorage
  components/              ← كل قسم من أقسام الصفحة في ملف منفصل
```

## تعديل المحتوى

كل النصوص (العناوين، المشاريع، الأسئلة الشائعة...) موجودة في ملف واحد:
`src/i18n/translations.ts` — عدّل النص هناك لكل من `ar` و `en` وسيظهر التغيير مباشرة.

## المشاريع المعروضة

المشاريع المعروضة (ATHR، Antika Gallery، قصتك على طريقتك، جملة أبو علي، نظام المخزن) مبنية على مشاريعك الفعلية.
لإضافة مشروع جديد، أضف عنصرًا جديدًا لمصفوفة `projects.items` في كل من `ar` و `en`.

## نموذج التواصل

نموذج التواصل شغال فعليًا بدون أي خادم: عند الإرسال، بيفتح واتساب في تاب جديد برسالة جاهزة فيها كل بيانات العميل، والعميل يضغط إرسال من عنده وخلاص.
تحت الفورم كمان أزرار تواصل مباشر (واتساب، تليجرام، إيميل) لمين مش عايز يملأ الفورم.

## أرقام وروابط التواصل

كل أرقام وروابط التواصل (واتساب، تليجرام، فيسبوك، جيت هاب، الإيميل) في مكان واحد: `src/config/contact.ts`
عدّل هناك بس، والتغيير هيظهر في كل الموقع (الفوتر + الفورم) تلقائيًا.

⚠️ **لسه محتاج تعدّلهم بنفسك:**
- `facebookUrl` — لسه رابط placeholder.
- `githubUrl` — لسه رابط placeholder.

## الألوان والخطوط

نظام الألوان والخطوط معرّف في `tailwind.config.js`:
- `ink` أسود مائل للأزرق (خلفيات داكنة)
- `paper` أبيض دافئ (خلفية أساسية)
- `brass` ذهبي نحاسي (اللون المميز الوحيد)
- خط Tajawal للعربية، IBM Plex Sans للإنجليزية، IBM Plex Mono للتفاصيل التقنية (Labels)

## قبل النشر الفعلي

- استبدل `og-image.jpg` في `public/` بصورة حقيقية للمشاركة على وسائل التواصل.
- حدّث رابط `https://al-mo.dev/` في `index.html` و `sitemap.xml` بدومينك الفعلي.
- حدّث `facebookUrl` و `githubUrl` في `src/config/contact.ts` بروابطك الفعلية.
- شغّل Lighthouse للتأكد من تجاوز 95 في الأداء.
